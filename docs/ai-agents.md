# AI Agent Frameworks

A curated list of useful AI agent frameworks on GitHub for building autonomous AI agents.

## 1. LangChain

**URL:** https://github.com/langchain-ai/langchain

The platform for building reliable AI agents. LangChain provides a flexible framework for developing applications powered by large language models, with robust tooling for chaining components, memory management, and tool use.

- **Stars:** 100k+
- **Language:** Python
- **Use case:** General-purpose LLM applications, RAG, agents

---

## 2. CrewAI

**URL:** https://github.com/crewAIInc/crewAI

Framework for orchestrating role-playing, autonomous AI agents. CrewAI enables multiple agents to work together collaboratively, tackling complex tasks through coordinated intelligence.

- **Stars:** 30k+
- **Language:** Python
- **Use case:** Multi-agent systems, collaborative workflows, role-playing agents

---

## 3. Agno

**URL:** https://github.com/agno-agi/agno

The programming language for agentic software. Agno provides primitives for building, running, and managing multi-agent systems at scale with a focus on reliability and performance.

- **Stars:** 20k+
- **Language:** Python
- **Use case:** Multi-agent orchestration, production-grade agent systems

---

## 4. Browser Use

**URL:** https://github.com/browser-use/browser-use

Make websites accessible for AI agents. This library enables automation of web tasks by providing AI agents with the ability to interact with websites through a controlled browser interface.

- **Stars:** 25k+
- **Language:** Python
- **Use case:** Web automation, browser-based agents, end-to-end testing

---

## 5. Composio

**URL:** https://github.com/ComposioHQ/composio

Powers 1000+ toolkits, tool search, context management, authentication, and a sandboxed workbench to help build AI agents that turn intent into action. Great for connecting agents to external services.

- **Stars:** 15k+
- **Language:** TypeScript/Python
- **Use case:** Tool integration, agent actions, workflow automation

---

---

## 6. Superpowers

**URL:** https://github.com/obra/superpowers

An agentic skills framework & software development methodology that actually works. Focuses on creating reusable agent skills that can be composed together.

- **Stars:** 62k+
- **Language:** Shell/Agentic
- **Use case:** Agentic development methodology, skill composition

---

## 7. pi-mono

**URL:** https://github.com/badlogic/pi-mono

AI agent toolkit: coding agent CLI, unified LLM API, TUI & web UI libraries, Slack bot, vLLM pods. Comprehensive toolkit for building agentic applications.

- **Stars:** 17k+
- **Language:** TypeScript
- **Use case:** Coding agents, unified LLM API, agent tooling

---

## 8. Shannon

**URL:** https://github.com/KeygraphHQ/shannon

Fully autonomous AI hacker to find actual exploits in web apps. Achieved 96.15% success rate on the hint-free, source-aware XBOW Benchmark.

- **Stars:** 25k+
- **Language:** TypeScript
- **Use case:** Security testing, autonomous exploitation research

---

## 9. memU

**URL:** https://github.com/NevaMind-AI/memU

Memory system for 24/7 proactive agents. Enables persistent context and memory management for always-on agentic applications.

- **Stars:** 11k+
- **Language:** Python
- **Use case:** Long-running agents, memory/context management

---

## 10. Dexter

**URL:** https://github.com/virattt/dexter

An autonomous agent for deep financial research. Specialized agent focused on financial analysis and research tasks.

- **Stars:** 16k+
- **Language:** TypeScript
- **Use case:** Financial research, quantitative analysis agents

---

## 11. Claude Mem

**URL:** https://github.com/thedotmack/claude-mem

Claude Code plugin that automatically captures everything Claude does during coding sessions, compresses it with AI, and injects relevant context back into future sessions.

- **Stars:** 31k+
- **Language:** TypeScript
- **Use case:** Agent memory, session context persistence

---

## 12. HuggingFace Skills

**URL:** https://github.com/huggingface/skills

Skills catalog for HuggingFace agents - similar to OpenAI's skills approach but for the HF ecosystem.

- **Stars:** 6.7k+
- **Language:** Python
- **Use case:** HuggingFace agent extensions, tool catalog

---

## 13. Kimi CLI

**URL:** https://github.com/MoonshotAI/kimi-cli

Kimi Code CLI - Moonshot AI's CLI agent. Chinese AI lab making strong progress on coding agents.

- **Stars:** 6.8k+
- **Language:** Python
- **Use case:** Coding agent, CLI tool

## Key Trends (Feb 2026)

1. **Agentic Skills** - Framework-agnostic skill systems (Superpowers, HF Skills, OpenAI Skills)
2. **Memory/Context** - Growing focus on persistent memory for long-running agents (memU, Claude Mem)
3. **Specialization** - Domain-specific agents (Shannon = security, Dexter = finance)
4. **TypeScript Dominance** - Most new agent frameworks are TypeScript-first
5. **Autonomous Persistence** - 24/7 agents becoming viable with proper memory management
