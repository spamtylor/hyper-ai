# AI Agent Frameworks

A curated list of useful AI agent frameworks on GitHub for building autonomous AI agents.

## 1. LangChain

**URL:** https://github.com/langchain-ai/langchain

The platform for building reliable AI agents. LangChain provides a flexible framework for developing applications powered by large language models, with robust tooling for chaining components, memory management, and tool use.

- **Stars:** 100k+
- **Language:** Python
- **Use case:** General-purpose LLM applications, RAG, agents

---

## 2. CrewAI

**URL:** https://github.com/crewAIInc/crewAI

Framework for orchestrating role-playing, autonomous AI agents. CrewAI enables multiple agents to work together collaboratively, tackling complex tasks through coordinated intelligence.

- **Stars:** 30k+
- **Language:** Python
- **Use case:** Multi-agent systems, collaborative workflows, role-playing agents

---

## 3. Agno

**URL:** https://github.com/agno-agi/agno

The programming language for agentic software. Agno provides primitives for building, running, and managing multi-agent systems at scale with a focus on reliability and performance.

- **Stars:** 20k+
- **Language:** Python
- **Use case:** Multi-agent orchestration, production-grade agent systems

---

## 4. Browser Use

**URL:** https://github.com/browser-use/browser-use

Make websites accessible for AI agents. This library enables automation of web tasks by providing AI agents with the ability to interact with websites through a controlled browser interface.

- **Stars:** 25k+
- **Language:** Python
- **Use case:** Web automation, browser-based agents, end-to-end testing

---

## 5. Composio

**URL:** https://github.com/ComposioHQ/composio

Powers 1000+ toolkits, tool search, context management, authentication, and a sandboxed workbench to help build AI agents that turn intent into action. Great for connecting agents to external services.

- **Stars:** 15k+
- **Language:** TypeScript/Python
- **Use case:** Tool integration, agent actions, workflow automation
