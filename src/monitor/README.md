# Hyper Monitor

## Real-time Monitoring
Track Hyper and sub-agents in real-time.

## Commands
```bash
# Check active agents
monitor agents

# Check task queue
monitor queue

# Check system load
monitor system

# Self-healing check
monitor heal
```

## Self-Healing
- Restart failed processes
- Clear stuck sessions
- Auto-retry failed tasks
