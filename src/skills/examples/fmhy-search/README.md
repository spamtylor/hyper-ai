# FMHY Search Skill

Search FMHY (Free Media Holy Land) for free resources, tools, and alternatives.

## Usage

```bash
./run.sh <search-query>
```

## Examples

```bash
./run.sh python tutorials
./run.sh video editing
./run.sh "audio books"
```

## Output

Saves research notes to `../../research/fmhy/` with timestamp.

## About FMHY

FMHY is a curated list of free alternatives to paid software and media resources.
- Website: https://fmhy.net
- Reddit: r/FREEMEDIAHECKYEAH
