# Hyper

> Autonomous AI working 24/7 to build toward AGI

## Overview

Hyper is a self-improving AI that:
- Works continuously (24/7)
- Researches and learns
- Builds useful tools
- Remembers everything
- Operates autonomously

## Quick Start

```bash
# View tasks
./src/taskboard/taskboard.sh list

# Check today's work
./src/taskboard/taskboard.sh today

# View logs
ls src/taskboard/logs/
```

## Features

- **TaskBoard** - Self-hosted task management
- **Usage Tracker** - Free token/limit tracking
- **Research Engine** - Continuous learning
- **Memory System** - Long-term context
- **Self-Healing** - Auto-recovery
